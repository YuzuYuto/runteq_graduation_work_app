## サービス概要
自分の好きなお笑い芸人のネタのおすすめを投稿して、自分の面白いを他者と共有し、さらに他者が共有したネタのおすすめを閲覧することができたり、コメントやいいねを通じて交流することができるサービスです。


## このサービスへの思い・作りたい理由
私自身お笑いが大好きで、日頃からYouTubeでお笑い芸人のネタ動画を視聴しているのですが、視聴する動画のジャンルが偏っていたり、新しいジャンルの動画を視聴したくても、アップされている動画の数が膨大なのでなかなか新しいジャンルを開拓できないでいました。そこで他者のネタのおすすめや、自身のおすすめを共有できるサービスがあるとさらに様々なネタを知ることができ、さらにお笑い芸人のネタ動画を楽しめると思い、このサービスを考案しました。


## ユーザー層について
お笑いに関しては性別年齢問わず幅広く人気があるので、対象のユーザー層は10代から60代のお笑い好きの男女です。


## サービスの利用イメージ
お笑いのネタ動画が視聴したい時に、こちらのサービスを使っておすすめのネタ動画を調べて、ネタの概要などを読むことによって視聴意欲が湧き、今まで視聴したことのなかったようなジャンルを楽しめるようになります。


## ユーザーの獲得について
いいねやコメント機能、SNSでシェアできる機能を実装して、自分のおすすめしたいものを他者に評価してもらうために共有したくなるような仕組みを作り、ユーザー自身で拡散してもらえるようにしてユーザーを獲得します。


## サービスの差別化ポイント・推しポイント
芸人のランキングサイトなど芸人をまとめたサイトはありますが、コメント機能やいいね機能はなく、他者と交流できるサービスではないので、他者からの評価を得ることによって投稿意欲を促す仕組みになっていません。また、そもそもサイトの管理者が単独で更新しているものも多く、ジャンルが偏りがちになったり、人気があり知名度の高いネタのみであったりすることも多いです。私が作るサービスでは、自分の好きなネタについて投稿することができるので、まだあまり知名度はないがおもしろいネタに出会うことができたり、そのネタを投稿している人の投稿一覧を閲覧することで好きな系統のネタを探すことができたり、コメントや概要を見ることで、視聴意欲を掻き立てることができます。
